# 🧸 Toyove-India – High-Fidelity Children's Marketplace Ecosystem

A premium, enterprise-grade e-commerce application serving as the definitive technical blueprint for Toyove-India's curated retail infrastructure. This project manages high-fidelity product interactions, transitioning hardware-inspired design tokens into a seamless consumer-facing subscription and shopping economy.

**Live Demo**: [toyove-india-jhkr.vercel.app](https://toyove-india-jhkr.vercel.app/)

---

## 🏗️ Engineering Core Pillars

- **Aesthetic Integrity**: A strict design system leveraging the `#FDF3E7` cream palette and interactive dashed-border components to maintain "Playful Premium" visual parity with reference marketplaces.
- **Predictive Intelligence**: Implementation of a high-throughput search engine for sub-second product retrieval, featuring live thumbnails and category-based indexing.
- **Marketplace Logic**: A robust shared-state ecosystem for managing shopping carts, user session persistence (localStorage), and dynamic collection-based navigation.

---

## 🚀 Marketplace Ecosystem Features

### 🔐 Advanced Security & Identity
- **Profile Hub**: Personalized user dashboards displaying original registration data and email-verified account details.
- **Session Persistence**: Encrypted local storage simulation for JWT-style login tokens and user metadata.
- **Dynamic Onboarding**: Multi-step registration flow that redirects to high-fidelity login validation before granting dashboard access.

### 🧩 Intelligent Navigation & Search
- **Predictive Search Panel**: "Google-style" live search bar that transitions between global results and detailed product suggestions as the user types.
- **Mega-Menu Architecture**: High-availability navigation system featuring categorical thumbnails, sub-category grouping, and responsive layouts for Mobile/Tablet/Desktop.
- **Collection Mapping**: URL-driven dynamic page generation that synchronizes hero imagery and filter logic based on specific navigation paths.

### 🛒 Precision Cart & Product Management
- **Live Cart Logic**: Real-time quantity adjustments and subtotal calculations with interactive spring-motion deletions.
- **High-Fidelity Detail Pages**: Dynamic product pages featuring high-resolution galleries, related product carousels, and breadcrumb tracking.
- **Global Localization**: Top-tier utility bar providing real-time country switching (US, India, UK) and language selection (English, Hindi, etc.) with currency indicators.

## 💻 Standard Operating Procedures (SOPs)

### 1. Navigation & Compliance
- **Identity Enforcement**: High-fidelity login required for Account Access; data persists via local handshakes.
- **Responsive Protocol**: Three-tier header adaptation (Desktop, Tablet, Mobile) ensures zero UI breakage across breakpoints.

### 2. Product Lifecycle & Discovery
- **Predictive Handshake**: Search input triggers filtered sub-string matching logic instantly.
- **Asset Authorization**: Clicking "Add to Cart" or "Wishlist" triggers localized state updates in the shared ledger.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.x or later)
- npm (v9.x or later)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Kunalsahuji/Toyove_India.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Toyove_India/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server with HMR:
```bash
npm run dev
```
